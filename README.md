# FMT-MAT
File Monitoring Tool

## Requirements
Windows
Python 3 ver

## Install
```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Usage
Python file
```bash
python run.py
```
Binary file
```bash
Windows x86 : monitor_x86.exe
Windows x64 : monitor_x64.exe
```
